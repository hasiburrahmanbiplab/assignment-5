Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a way to represent an HTML or XML document as a tree structure. Each part of the webpage (like text, images, or buttons) is called a node, and JavaScript can interact with these nodes to change the content or style of the webpage dynamically.

How do you select an element from the DOM?

I can select an element from the DOM using methods like:


  1. 1. getElementById(): Selects an element by its id attribute.
  2. 2. getElementsByClassName(): Selects elements by their class name.
  3. 3. getElementsByTagName(): Selects elements by their tag name.
  4. 4. querySelector(): Selects the first element that matches a specified CSS selector.
  5. 5. querySelectorAll(): Selects all elements that match a specified CSS selector.
What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript where instead of attaching event listeners to each individual element, i attach a single event listener to a parent element. This listener can handle events for all child elements, even if they are added dynamically later.


Why is it useful?


  1. 1. Efficiency: Reduces the number of event listeners, improving performance.
  2. 2. Dynamic Elements: Works for elements added to the DOM after the page is loaded.
How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes, i can use methods to get or set the values of attributes like src, href, etc. For example, i can change an image's source or a link's URL.
To manipulate an element's styles, i can directly change its inline CSS styles, such as background color, font size, etc.
